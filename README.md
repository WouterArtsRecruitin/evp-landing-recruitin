# EVP Landing Page - Recruitin.nl

Employee Value Proposition landing page voor recruitment doeleinden.

## Setup
1. Run `./setup-evp-landing.sh`
2. Run `./create-landing-page.sh`
3. Run `./deploy.sh`

## Features
- Responsive design
- Modern UI/UX
- SEO optimized
- Fast loading
- Contact integration

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Netlify hosting
